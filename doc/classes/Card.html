    Makes a transaction to this card

    createTransaction(string $type, float $amount, string $description, string $merchant, boolean $force) : \Transaction

    Parameters

    $type

    string

    the type of transaction

    $amount

    float

    the amount to make a transaction for

    $description

    string

    the description of the transaction

    $merchant

    string

    the name of the merchant

    $force

    boolean

    if TRUE, doesn't check to make sure the card is active. If FALSE, then checks to make sure the card isn't active before making a transaction

    Exceptions

    \ResponseException if this card is inactive AND $force is false, OR if $amount causes the card balance to fall below zero.

    Returns

    \Transactionthe resulting Transaction object
